HeadsUp displays the output of a couple of commands in the bottom corners of your OSX Desktop.

Think GeekTool, but simpler.

== Dependencies

RubyCocoa. I expect everything to Just Work on Leopard, but I'm developing on Tiger with the MacPorts rb-cocoa package.

== Install

Stable releases are available at http://matthewtodd.github.com/heads_up.

Development versions can be tracked like so:

  $ git clone git://github.com/matthewtodd/heads_up.git
  $ cd heads_up && rake
